# Scientific Math & Data Playground 🧪

> **"THIS IS AN EXPERIMENTAL REPO"**

This is my personal, experimental repository for learning and practicing scientific mathematics and data analysis. I am building hands-on scripts and programs to understand mathematical concepts under the hood using both **Python** and **C++**.

## What’s Inside 📂
* **Linear Algebra:** Coding matrix operations, scalars, and vector manipulations.
* **Numerical Analysis:** Implementing data interpolation methods and curve fitting.
* **Data Analysis:** Writing scripts to process, clean, and analyze datasets.

## Getting Started 🚀
You will need a modern C++ compiler (`g++`) and a Python 3 environment.

### Run Python Scripts
1. **Run directly:** `python script_name.py`

### Run C++ Programs
1. **Compile:** `g++ -std=c++17 -o output_name file_name.cpp`
2. **Run:** `./output_name`

## Learning Goals 🎯
* Bridge the gap between mathematical theory and code implementation.
* Master data manipulation in both a high-level language (**Python**) and a system-level language (**C++**).
* Learn to optimize math formulas for computation.

*Note: This is a self-study space and not a professional production-grade repository. Code might be messy, but the learning is real!*
